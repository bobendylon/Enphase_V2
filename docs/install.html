Enphase V2

Installation du firmware directement dans le navigateur

Prérequis

  • Chrome ou Edge (desktop)
  • ESP32-S3 DevKit connecté en USB
  • Aucun téléchargement : le firmware est chargé en ligne et flashé directement

Installation

Cliquez sur le bouton puis suivez les instructions. Le firmware sera téléchargé depuis GitHub et flashé sur votre ESP32.

Le firmware provient de la dernière version publiée sur GitHub Releases. Aucun fichier n’est enregistré sur votre ordinateur.
